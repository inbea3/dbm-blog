<h2>JavaScript异步编程</h2><p>异步编程是JavaScript的核心特性之一。本文详细介绍Promise和async/await的使用方法，帮助你更好地处理异步操作。</p><h3>为什么要用异步编程？</h3><p>JavaScript是单线程的，如果所有操作都是同步的，长时间运行的任务会阻塞主线程，导致页面卡顿。异步编程可以解决这个问题。</p><h3>Promise基础</h3><pre><code>const fetchData = () => {
  return new Promise((resolve, reject) => {
    setTimeout(() => {
      resolve('数据加载成功');
    }, 1000);
  });
};</code></pre><h3>async/await</h3><p>async/await是Promise的语法糖，让异步代码看起来像同步代码：</p><pre><code>async function getData() {
  try {
    const result = await fetchData();
    console.log(result);
  } catch (error) {
    console.error(error);
  }
}</code></pre>
